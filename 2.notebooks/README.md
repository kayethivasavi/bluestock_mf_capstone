# Bluestock Mutual Fund Analytics Capstone

## Project Overview
This project analyzes mutual fund data using Python, SQL, and dashboards.

## Features
- Data cleaning
- Database creation
- EDA analysis
- Risk analysis (VaR, Sharpe)
- Investor analysis
- Dashboard creation

## How to run
Install packages:
pip install pandas numpy matplotlib plotly

Run database:
python create_database.py

Open notebook:
python -m notebook

Run dashboard:
python 2.notebooks/dashboard_day5.py

## Output
- Reports in 5.reports folder
- HTML dashboards
- CSV analysis files
